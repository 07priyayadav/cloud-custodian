.. _gcp.router.actions.delete:

delete
++++++
`Deletes <https://cloud.google.com/compute/docs/reference/rest/v1/routers/delete>`_ a router

:Example:

.. code-block:: yaml

    policies:
      - name: gcp-network-unattached-routers
        description: Deletes unattached Cloud Routers
        resource: gcp.router
        filters:
           - type: value
             key: interfaces
             value: absent
        actions:
           - delete

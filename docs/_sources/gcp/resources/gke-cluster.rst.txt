.. _gcp.gke-cluster.actions.delete:

delete
++++++
Action to delete GKE clusters

It is recommended to use a filter to avoid unwanted deletion of GKE clusters

:example:

.. code-block:: yaml

        policies:
          - name: gcp-delete-testing-gke-clusters
            resource: gcp.gke-cluster
            filters:
              - type: value
                key: name
                op: regex
                value: '^(test-|demo-)*'
            actions:
              - type: delete

.. _aws.alarm.actions.delete:

delete
++++++
Delete a cloudwatch alarm.

:example:

.. code-block:: yaml

        policies:
          - name: cloudwatch-delete-stale-alarms
            resource: alarm
            filters:
              - type: value
                value_type: age
                key: StateUpdatedTimestamp
                value: 30
                op: ge
              - StateValue: INSUFFICIENT_DATA
            actions:
              - delete

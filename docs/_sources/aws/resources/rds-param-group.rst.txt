copy
++++
Action to copy an RDS parameter group.

:example:

.. code-block:: yaml

        policies:
          - name: rds-param-group-copy
            resource: rds-param-group
            filters:
              - DBParameterGroupName: original_pg_name
            actions:
              - type: copy
                name: copy_name
